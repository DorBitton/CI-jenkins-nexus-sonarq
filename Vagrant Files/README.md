## Vagrant

 <img src="" alt="Terminal">


- Steps for installing Vagrant:
 *Update system:
```
sudo apt update
sudo apt upgrade
```
 *Install VMWare:
```
sudo apt install virtualbox
```
 *Install Vagrant:
```
$ curl -O https://releases.hashicorp.com/vagrant/2.2.9/vagrant_2.2.9_x86_64.deb

$ sudo apt install ./vagrant_2.2.9_x86_64.deb
```
